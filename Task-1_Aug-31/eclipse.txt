3. Walk me through the steps to download and install Eclipse IDE

->  Steps to download and install Eclipse IDE:
    1.  Go to Eclipse Website https://www.eclipse.org/.
    2.  Choose the appropriate package like "Eclipse IDE for Java Developers".
    3.  Click the download link and choose a mirror.
    4.  Extract the downloaded ZIP file, run the installer executable (e.g., eclipse-inst), 
        and select the downloaded IDE version.
    5.  Choose Installation Folder
    6.  Specify workspace location for projects.
    7.  Click "Install" to begin installation.
    9.  Configure workspace preferences if prompted.
